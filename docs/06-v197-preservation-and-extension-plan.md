# V197 Preservation and Extension Plan

## Immutable source

Canonical host SHA-256 must remain `252eee806ece31ef829a2dc5cd45aa8d8f8e855db1bde98b6f87193d786633c3`. Decoded Entry and Universe hashes remain integrity gates. Source bytes are copied/verified, not edited.

## Allowed extension

- one module bridge loader appended only in the served response;
- event delegation and idempotent listener cleanup;
- read/write hydration into existing semantic slots;
- bridge-owned style nodes identified by stable IDs;
- V197-native popovers, detail chambers, and adjunct documents when no native slot exists;
- measured geometry fixes that do not replace visual identity.

## Immediate visual work

- re-measure wordmark, master star, Creation, system labels, legend, and Add System at desktop/mobile;
- add custom top-bar language trigger without raw `<select>` presentation;
- keep map node/detail geometry collision-free;
- pause heavy offscreen animation, tier particles/effects, and avoid duplicate loops/listeners;
- test reduced motion and real WebKit.

## Regression gates

Exact hashes, no `#root`, no React geometry CSS, no forbidden spellings, no white native controls, no staged primary route, no hidden unregistered control, no DOM overlap, no document overflow, and screenshots from Chromium plus genuine WebKit.
