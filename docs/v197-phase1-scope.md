# V197 Phase 1 Scope

Phase 1 restores only canonical V197 presentation authority and read-only backend hydration.

Included:

- byte-checked canonical V197 host under `apps/web/public/v197/`;
- direct V197 product-route composition: canonical host plus one nonvisual bridge loader;
- no visible React root or React geometry CSS;
- route synchronization across existing native Today, Talk, Journal, Plan, Systems, Map, Orbits, Timeline, Insights, Research, Community, and Web Signals controls;
- read-only session, preferences, owner-state, map, orbit, timeline, and insight hydration into existing V197 text slots;
- bridge scaffold event names and API/stream seams;
- desktop, mobile, and WebKit parity proof.

Explicitly excluded:

- write bindings;
- V197-native Capsule, Settings, and Omega adjunct documents;
- new Capsule, Settings, or Omega visual DOM;
- Talk streaming, OpenAI interaction changes, research providers, community providers, and new backend migrations.

Phase 1 may only be called `PHASE_1_PASS`. It is not full-product acceptance.
